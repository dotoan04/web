Trong phần này, chúng ta sẽ tìm hiểu về các khái niệm cơ bản.

### Ví dụ code

Đây là cách bạn có thể thêm code block:

```javascript

  console.log("Hello, World!");
  return true;
}

// Gọi function
helloWorld();
```

### Ví dụ code Python

```python
def fibonacci(n):
    """Tính số Fibonacci thứ n"""
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# Test
print(fibonacci(10))
```

## Metadata fields giải thích:

**Frontmatter (phần giữa --- và ---):**
- `title`: Tiêu đề bài viết (bắt buộc)
- `slug`: URL slug (tự động tạo nếu để trống)
- `excerpt`: Mô tả ngắn cho SEO và danh sách bài viết
- `status`: DRAFT (nháp) | PUBLISHED (xuất bản) | SCHEDULED (hẹn giờ)
- `categoryId`: ID của chuyên mục (lấy từ admin/categories)
- `tagIds`: Mảng các ID thẻ (lấy từ admin/tags)
- `coverImageId`: ID của ảnh bìa (upload tại admin/media)
- `publishedAt`: Thời gian xuất bản (format: YYYY-MM-DDTHH:mm)
- `author`: Tên tác giả (tùy chọn)
- `date`: Ngày viết (tùy chọn)

**Lưu ý:**
1. Để lấy categoryId: Vào Admin → Categories, copy ID của chuyên mục
2. Để lấy tagIds: Vào Admin → Tags, copy các ID thẻ và thêm vào mảng: ["id1", "id2"]
3. Để lấy coverImageId: Vào Admin → Media, upload ảnh và copy ID
4. Nếu status là SCHEDULED, nhớ điền publishedAt
5. Các field để trống sẽ sử dụng giá trị mặc định

**Tips viết markdown:**
- Sử dụng heading levels: # (h1), ## (h2), ### (h3)
- Code blocks hỗ trợ nhiều ngôn ngữ: javascript, python, typescript, html, css, v.v.
- Có thể thêm inline code với backticks: `code`
- Thêm liên kết: [text](url)
- Thêm hình ảnh: ![alt](url)
- Danh sách: - hoặc * cho không thứ tự, 1. 2. 3. cho có thứ tự
- Trích dẫn: Bắt đầu dòng với >
